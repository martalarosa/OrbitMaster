# 🛰 Моделирование движения НИСЗ с гравитационными возмущениями

Программно-математическое обеспечение (ПМО) по лабораторной работе №2  
**"Математическое моделирование движения навигационных искусственных спутников Земли с учётом гравитационных возмущений"**, МАИ, кафедра 704

## 🎯 Цель
Разработать ПМО для численного анализа движения НИСЗ в центральном поле Земли с возможностью включения гравитационных возмущений от Луны, Солнца, Юпитера и солнечного ветра. 
Реализовать выбор моделей, построение траектории, графиков координат и скоростей, а также обеспечить физическую корректность.

## ⚙️ Возможности
- расчёт движения НИСЗ методом численного интегрирования (solve_ivp)
- учёт гравитационных возмущений:
  - Луна
  - Солнце
  - Юпитер
  - Солнечный ветер (постоянное ускорение)
- режим физически корректной модели (учёт реакции Земли)
- выбор возмущений через графический интерфейс
- графики X(t), Y(t), Z(t), Vx(t), Vy(t), Vz(t) + 3D-орбита
- GUI-интерфейс на `tkinter`
- автоматические юнит-тесты
- интеграция с CI/CD (GitHub Actions и GitLab CI)

## 🖼 Интерфейс
- Ввод начальных условий (координаты, скорости, шаг, время)
- Выбор возмущений (из выпадающего списка)
- Переключатель физической корректности (галочка)
- Кнопка запуска моделирования
- Вывод трёх графиков: координаты, скорости, 3D-орбита

## 🧪 Тестирование
Тесты покрывают:
- поведение базовой гравитации Земли
- влияние Луны, Солнца, Юпитера
- эффект солнечного ветра
- различие между демонстрационным и физически корректным режимом

**Запуск вручную:**
```bash
pytest -v test_lab2.py
```
